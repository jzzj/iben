iben.js框架作为辅助性框架。
src文件夹下的文件为最新版本，不提供压缩版本。
iben采取沙箱模式加载模块。
通过增加伪原生对象的方式，给原型增加属性和方法，分别为：iBen.Array,iBen.String,iBen.Number,iBen.Date。
DOM操作及事件添加等操作采用jQuery的方式，部分会使用原始对象。
更多的是为提供一种开发约束及开发模式。
该库中会设计多种模式：观察者、装饰者、工厂等模式。

改库主要为了控制项目框架，代码结构，提供模块化编程。更容易的调试及代码间的松散耦合。


iben.html为iben框架中的功能调试及提供的API使用方式。


iBen中的log方法只是一个伪log方法，实际调用的还是console.log，同样会存在浏览器兼容问题，不过不会报错。time方法同理。
